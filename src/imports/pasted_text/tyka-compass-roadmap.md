Améliorations prioritaires de TYKA Compass – Expérience utilisateur, qualité du diagnostic et suivi des évaluations

La nouvelle version de TYKA Compass doit évoluer vers un véritable système intelligent d'orientation, de diagnostic et d'accompagnement personnalisé. Plusieurs améliorations sont nécessaires afin de renforcer la qualité de l'expérience utilisateur, la pertinence des résultats et le suivi de l'évolution des membres dans le temps.

1. Amélioration de l'expérience utilisateur pendant les évaluations
Navigation dans les questionnaires

Actuellement, lorsqu'un utilisateur avance dans le questionnaire, il ne peut plus revenir sur ses réponses.

Ajouter impérativement :

Bouton "Précédent"

L'utilisateur doit pouvoir :

revenir à la question précédente ;
modifier une réponse déjà donnée ;
naviguer librement dans une même évaluation avant la soumission finale.
Sauvegarde automatique

Le système doit enregistrer automatiquement les réponses au fur et à mesure afin d'éviter toute perte de données en cas de coupure Internet ou de fermeture accidentelle de la page.

Barre de progression

Afficher :

le pourcentage de progression ;
le nombre de questions restantes ;
l'estimation du temps restant.

Exemple :

Progression : 65 % – 12 questions restantes – environ 4 minutes.

2. Renforcement de la qualité des résultats

Les rapports générés doivent redevenir beaucoup plus analytiques et personnalisés.

Chaque rapport doit obligatoirement contenir :

Résumé exécutif

Présentation synthétique du profil de l'utilisateur.

Points forts

Identification automatique des dimensions les mieux notées.

Pour chaque point fort :

score obtenu ;
interprétation ;
impact potentiel sur le parcours académique, professionnel ou entrepreneurial.
Axes prioritaires d'amélioration

Identification des dimensions les plus faibles.

Pour chaque axe :

diagnostic ;
risque associé ;
recommandations pratiques ;
ressources recommandées.
Recommandations intelligentes

Les recommandations doivent être contextualisées selon :

l'âge ;
le niveau académique ;
le statut professionnel ;
le projet professionnel ;
le secteur d'activité ;
le type de diagnostic réalisé.

Les recommandations doivent être concrètes, détaillées et directement applicables.

3. Renforcement du module Orientation Académique

Le module Orientation Académique doit être développé comme un véritable conseiller d'orientation.

Analyse du parcours actuel

Prendre en compte :

diplôme actuel ;
série du BAC ;
licence obtenue ;
master obtenu ;
spécialité actuelle ;
expérience professionnelle ;
projet professionnel visé.
Analyse de cohérence

Calcul automatique d'un :

Score de cohérence académique

avec interprétation :

Très cohérent
Cohérent
Partiellement cohérent
Faiblement cohérent
Incohérent
Analyse des écarts

Le système doit identifier :

les compétences manquantes ;
les prérequis académiques absents ;
les expériences à acquérir ;
les certifications pertinentes.
Proposition de trajectoire académique

Le système doit générer automatiquement :

Parcours recommandé

Exemple :

BAC D
→ Licence Informatique
→ Master Intelligence Artificielle
→ Certification Data Science
→ Métier visé : Data Scientist

Formations complémentaires recommandées

Le système doit suggérer :

certifications professionnelles ;
MOOCs ;
formations TYKA ;
ateliers ;
conférences ;
programmes d'incubation ;
clubs et communautés pertinentes.
4. Création d'un espace Historique des évaluations

Créer dans l'espace utilisateur une rubrique :

Historique TYKA Compass

Cette rubrique permettra au membre de suivre son évolution dans le temps.

Tableau unique des évaluations

Créer un tableau consolidé regroupant toutes les évaluations réalisées.

Colonnes suggérées :

Date	Type d'évaluation	Score global	Niveau	Évolution
10/01/2026	Développement personnel	68 %	Intermédiaire	+8 %
05/03/2026	Développement professionnel	72 %	Bon niveau	+4 %
20/05/2026	Orientation académique	80 %	Très cohérent	+12 %
Filtres

Permettre de filtrer l'historique :

par période ;
par année ;
par trimestre ;
par mois ;
par type d'évaluation.
Visualisation graphique

Ajouter des graphiques d'évolution.

Évolution globale

Courbe montrant la progression générale du membre.

Évolution par rubrique

Graphiques distincts pour :

Développement personnel ;
Développement professionnel ;
Entrepreneuriat ;
Orientation académique ;
Renforcement d'activité ;
Projet professionnel.
Comparaison entre évaluations

Permettre à l'utilisateur de comparer deux évaluations.

Exemple :

Évaluation du 01/01/2026
VS
Évaluation du 01/06/2026

Le système affiche :

les compétences ayant progressé ;
les compétences stables ;
les compétences en régression ;
les nouvelles recommandations.
5. Tableau de bord de progression personnelle

Créer un espace :

Mon évolution TYKA

Affichage :

Score global actuel
Progression depuis la première évaluation
Compétences les plus améliorées
Compétences nécessitant encore un renforcement
Prochaines recommandations prioritaires
Objectifs suggérés pour les 90 prochains jours
6. Rapport PDF amélioré

Le rapport PDF doit inclure :

Logo TYKA ;
Date d'évaluation ;
Identifiant du rapport ;
Résumé exécutif ;
Graphique radar ;
Points forts ;
Axes d'amélioration ;
Recommandations personnalisées ;
Orientation académique (si concernée) ;
Comparaison avec l'évaluation précédente ;
Plan d'action personnalisé.
Objectif final

Faire de TYKA Compass un véritable coach numérique intelligent, capable non seulement d'évaluer la situation actuelle d'un utilisateur, mais aussi de suivre sa progression dans le temps, mesurer ses évolutions, identifier ses besoins prioritaires et lui proposer des recommandations personnalisées en matière de développement personnel, professionnel, entrepreneurial et académique.
